This project needs an external server of playfab to work as a multiplayer game 
Please add a PlayFab account 
Do server build from server scene 


Commands for building server 
Install docker 
Linux 
and WSL for windows 
please go through building game manager server docs from playfab

 

docker login -u "UserName Here"


sudo docker build -t  "UserName Here"/"server name"."Version" 
sudo  docker login "UserName Here"
sudo docker push  "UserName Here"/"server name"."Version"
